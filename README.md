# multi-social-share-js
Simple dependency free multi social share script

# Supported share types
- Facebook
- Twitter
- Google
- Pinterest
- LinkedIn
- BufferApp
- Tumblr
- Reddit
- StumbleUpon
- Evernote
- Email
- Wordpress
- Pocket

# Usage
```javascript
Share.facebook();
```
### Button

```html
<button onclick="Share.facebook()">Share</button>
```

### Anchor tag
```html
<a href="" onclick="Share.facebook(); return false;">Share</a>
```
